## wsh v0.42.0 Release Notes

### Mobile Support
Touch-optimized UI with inertia scrolling, swipe gestures, adaptive layout, and virtual keyboard handling.

### Inline Agent Chat
Run skills inline within catalog cards — streamed output, draft persistence, and live status indicators without leaving the page.

### Skills Cards
New card-based catalog with input prompts, active session indicators, and keyboard shortcuts (Enter to run, Cmd+Enter for new tab).

### AI Skills Installer
Install skills from ai-skills.bytedance.net directly from the catalog — paste a URL and go.
